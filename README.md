# 项目说明
这是登录功能的前端实现 \
主要页面有
- 1 主界面（导航栏，内容栏，脚标栏）
- 2 登录功能页面
- 3 注册功能页面
- 4 错误页面，统一错误处理
- 5 home界面，有一些小设计（需要在第一个输入框输入 我要登录 可以进入welcomePage页面）
- 6 管理员和用户的欢迎界面（内含权限按钮等）
- 7 使用全局环境地址管理，根目录下.env文件
- 8 使用vuex进行内存管理，使用router进行路径处理，使用element-ui进行大部分界面设计
- 9 api等专门进行封装，体现分模块设计理念

# 使用说明
## Project setup 项目设置
```
npm install
```
### Compiles and hot-reloads for development 开发情况使用这个
```
npm run serve
```
### Compiles and minifies for production 为生产环境进行编译压缩
```
npm run build
```
### Lints and fixes files
```
npm run lint
```
